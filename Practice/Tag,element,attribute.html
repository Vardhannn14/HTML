Tags and types of Tags

There are 2 types of tags.

a.Which comes in pair.(starting tag and Closing)

eg:(p for paragraph/ h1-h6 for heading/title for title etc)

b.Which are self closing.(Self close/Empty/Void)

eg:(hr-horizontal line between two elements/ br-break line/ etc)

Element

Elements is nothing but Start Tag-Content-Close Tag

Attribute & Attribue Value

Attribute provides us information,after that the behaviour of element will changes.
